# Mini CRUD API com PHP puro

## Descrição

Um projeto de estudo.

**Objetivo:**  
&nbsp;&nbsp;&nbsp;&nbsp;• Criar uma API REST simples sem frameworks

&nbsp;&nbsp;&nbsp;&nbsp;• Respostas em JSON  

**Tecnologias:** PHP + MySQL
